## ANGR - A Symbolic Execution Tool for Binaries 

ANGR is a binary analysis platform that can conduct symbolic execution. 

> https://docs.angr.io/en/latest/quickstart.html


### Tasks For You To Do 

- Create a Python program that uses the [ANGR API](https://docs.angr.io/en/latest/quickstart.html) to report the following for each of the binary files in the `binaries.tar.gz` file. 
  - Function address
  - Function name 
  - ELF objects 
  - Memory map range for each identified elf object

- Code for the task: 50%

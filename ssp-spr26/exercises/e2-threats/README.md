# Using generative AI tools for threat modeling 

## Dependencies 

- Access to the Internet via a browser 


### Tasks For You To Complete
- Open up Claude and Gemini 
- Use chain of thought prompting to derive 5 threats models for [etcd](https://etcd.io/). You have to construct your prompt. 

## Deliverables [Rubric]

- All threats reported by Claude and Gemini [20%]
- All post conditions reported by Claude and Gemini [20%]
- All threat actors reported by Claude and Gemini [20%]
- All preconditions reported by Claude and Gemini [20%]
- Rank of the identified threats reported by Claude and Gemini [20%]


## Deliverables 

- All output in one text file or multiple text files in a zip file. 
- The prompt that you used in a text file
